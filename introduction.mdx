## Overview

VetyCare is a modern Android mobile application designed for veterinary care management. Built with Kotlin and leveraging Material Design 3, it provides a clean, intuitive interface for managing veterinary services.

## Technology Stack

VetyCare is built with modern Android development tools and libraries:

- **Language**: Kotlin
- **UI Framework**: Material Design 3
- **Architecture Components**: Android Jetpack (Navigation, AppCompat, ConstraintLayout)
- **Build System**: Gradle with Kotlin DSL
- **Minimum SDK**: Android 9.0 (API 28)
- **Target SDK**: Android 14+ (API 36)
